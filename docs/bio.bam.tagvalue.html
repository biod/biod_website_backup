Get tag for type T.

 Value v = "zzz";
 assert(v.tag == GetTypeId!string);

struct  Value;

Struct for representing tag values.

Tagged union, allows to store 8/16/32-bit integers, floats, chars, strings, and arrays of integers/floats.

Currently, opCast is very restrictive and requires that the requested type is exactly the same as stored in  Value (otherwise, ConvException is thrown). That means that you can't cast  Value to string when it contains integer, although it's possible to convert integer to string.
